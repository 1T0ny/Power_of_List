# Power_of_List

Create a function to that can compute large exponents (mn) by means of storing the result in a
linked list. Create an UI for users to enter the values for m and n. Display the result as well as
store the result in an external text file. Also present runtime analysis at each request. Perform
rigorous testing to find its limits and limiting factors. Implement input validation to ensure valid
and reasonable bounds for m and n.
